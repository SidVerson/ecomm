# ECOMM STORE

This store is made by express js

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

## Features

-Cookie-based authentification
-Shopping cart
-Hashing users data
